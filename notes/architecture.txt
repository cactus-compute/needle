Co-Designed Architecture & Training Recipe for Small-Scale Tool Calling
=======================================================================

## 1. Thesis

No single component is novel in isolation. The contribution is a
co-designed system: an encoder-decoder architecture trained with a
multi-objective recipe (generation + contrastive + Matryoshka +
structured weighting + QAT + pruning) that produces a small model
competitive with much larger decoder-only models on tool calling.

The architecture choices are interdependent:
  - We do not need MLPs for tiny models 
  - Structured loss weighting targets the actual error distribution 
  - QAT provides regularisation eliminating the need for dropout 
  - Attention is a nonlinearity in the absence of MLP 

----

Our blocks (with no_feedforward=True) are essentially:                                                                                                                                                                                              

  # encoder:                                                                                                                                                                                                                             
  x = x + Attention(Norm(x))

  # decoder:                                                                                                                                                                                                  
  x = x + SelfAttn(Norm(x))                                                                                                                                                                                                         
  x = x + CrossAttn(Norm(x))                                                                                                                                                                                                        
                                                                                                                                                                                                                                                       
The residual connection is the only pathway preserving information between layers. 
Without FFN, there's no nonlinear feature transformation, each layer just incrementally mixes information via attention and adds it back.                        
                                                                                                                                                                                                                                                      
The argument for Kimi-style residual bypass here would be: if you remove the residual, each attention layer gets to fully rewrite the representation rather than just adding a delta. 
In a standard transformer with FFN, the FFN does the heavy "rewriting" work and the attention residual is fine as an additive refinement. 
But in our attention-only setup, forcing attention into an additive role may be limiting, it can only nudge the representation, never reshape it.                 
Without FFN and without residuals, we lose the gradient highway entirely. Deep networks would struggle to train. 
Kimi's technique likely works in models that still have FFN layers (or some other nonlinearity) providing an alternate signal path.

With the gated residual, each line becomes:

  # gated residual — each sublayer gate is a learnable scalar, initialized to 0
  # sigmoid(0) = 0.5, so training starts as half-strength standard residual
  # gate → 1 recovers the standard residual exactly

  # encoder (per layer)
  g = sigmoid(gate)
  x = x + g * Attention(Norm(x))

  # decoder (per layer)
  g_s = sigmoid(self_attn_gate)
  g_c = sigmoid(cross_attn_gate)

  x = x + g_s * SelfAttn(Norm(x))
  x = x + g_c * CrossAttn(Norm(x))

The full residual is always preserved. The gate only scales the sublayer contribution.
At init (gate=0, g=0.5): half-strength sublayer addition.
gate → 1 (g=1): standard residual. gate → -inf (g=0): identity, sublayer ignored.
